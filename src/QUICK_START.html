Quick Start Guide - Get up and running in 5 minutes!

1 Install XAMPP or WAMP

Download and install XAMPP or WAMP on your computer. These provide Apache, MySQL, and PHP.

Download: XAMPP or WAMP

2 Start Services

Open XAMPP/WAMP control panel and start these services:

  • Apache - Web server
  • MySQL - Database server

Make sure both show green/running status.

3 Place Project Files

Copy this entire project folder to your web server directory:

XAMPP: C:\xampp\htdocs\field-maintenance\ WAMP: C:\wamp64\www\field-maintenance\

4 Create Database

1. Open phpMyAdmin by clicking the button below

2. Go to "SQL" tab

3. Copy and paste the entire contents of /database/create_database.sql

4. Click "Go" to execute

5 Test & Configure

Use the buttons below to test your setup and configure if needed.
